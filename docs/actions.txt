Действия по добавлению Redis в db-service:


1. Добавить redis  в docker-compose.yml
2. добавить пакет github.com/redis/go-redis/v9
3. создам файл redis.go
4. туда добавлю клиента-redis 
5. в каждом методе в task_server.go реализовать кэширование
    1. для гет запроса смотрим кэш, если там нет идем в бд и сохраняем в кэш
    2. при изменении задачи обновляем кэш (get, set, del)
    3. при удалении чистим кэш
6. добавить инициализацию Redis в main.go
